# Company Website

Phase-1 company website for a Bulgaria-based B2B software consulting business.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- MDX for insights/articles
- Vercel-ready hosting

## Local Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content To Customize Before Launch

- Replace contact email/domain in `src/data/site.ts`
- Add official company registration details after incorporation
- Replace or expand anonymized work examples with real case studies
- Connect the contact form to Resend or another backend when ready
- Set `NEXT_PUBLIC_SITE_URL` to the production domain
